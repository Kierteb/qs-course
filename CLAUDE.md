# QS Learning Platform

## Project Overview
A personal interactive QS course website hosted on GitHub Pages.
Static site: HTML, CSS, vanilla JavaScript only. No frameworks.

## Tech Stack
- HTML5, CSS3, vanilla JS (ES6+)
- No build tools, no npm, no frameworks
- All progress stored in localStorage
- Hosted on GitHub Pages (root of repo)

## File Structure
- index.html — dashboard/home
- modules/module-XX.html — one file per module
- css/main.css — global styles
- css/module.css — module page styles
- js/progress.js — progress tracking and localStorage logic
- js/spaced-repetition.js — SR algorithm and question bank
- js/exercises.js — BoQ exercise and calculator logic
- assets/ — any icons or images

## Conventions
- All JS is vanilla, no jQuery, no frameworks
- CSS uses custom properties (variables) for all colours and spacing
- Mobile responsiveness is NOT required — desktop only
- Commit messages: descriptive, present tense
